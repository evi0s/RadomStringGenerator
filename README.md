# RandomStringGenerator
A Random String Generator

## Usage

### -n, --number
Number of strings

Default value: 100

### -l, --length
Length of each string

Default value: 1000

### -p, --path
Path of output file

Default value: ./testcase.in

### -i, --interactive
Interactive mode

### --help
Print the usage

## Other points

* run without any parm will generate as well, which using the default values.
